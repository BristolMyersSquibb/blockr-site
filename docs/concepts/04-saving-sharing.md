# Saving and sharing

A board saves as one unit: blocks, links, layout and visibility toggles.

- **Save** (navbar) stores the board under its name; the label next to it shows the save state.
- The **menu at the left of the navbar** opens the workflow list: **Workflows** reopens any saved board, **History** lists earlier saves of the current one.
- **New** starts a fresh board in place; the arrow next to it opens one in a new browser tab.

Storage is a pins board: local when you run blockr yourself, per-user on Posit Connect. On Connect the menu gains a **Sharing** tab to set a workflow's visibility and share it with other users.

Details and backend options: [blockr.session](https://github.com/BristolMyersSquibb/blockr.session).
